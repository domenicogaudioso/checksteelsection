1. Panoramica Generale

ProfiliAcciaio v1.5 è un'applicazione web interattiva per la classificazione e verifica formale di profili strutturali in acciaio, sviluppata per supportare l'ingegnere strutturista nelle verifiche di resistenza, instabilità locale e globale secondo le normative europee e italiane vigenti.

🚀 Motore di Calcolo

Implementazione rigorosa dell'Eurocodice 3 (EN 1993-1-1) per la progettazione di elementi strutturali in acciaio con formula chiuse e metodo LTBeam-style.

📊 Visualizzazione

Rendering 2D della geometria della sezione e sviluppo 3D della membratura tramite grafici interattivi.

📁 Gestione Progetti

Import/export dell'input in formato JSON e download di sintesi in formato CSV per la documentazione.

📚 Database Integrato

Libreria di sezioni standard europee: IPE, HEA, HEB, HEM, UPN, RHS, SHS, CHS, L, U, PFC.

2. Funzionalità e Verifiche Strutturali

L'applicazione esegue verifiche complete secondo l'approccioSLU (Stati Limite Ultimi) con il seguente flusso:

Flusso delle Verifiche Strutturali

Verifiche Implementate

⚖ Forza Normale - Sezione

Verifica della resistenza plastica/elastica a sforzo assiale. Formula: ηN,sez = NEd / Npl,Rd

⚖ Forza Normale - Instabilità

Verifica di instabilità flessionale attorno agli assi y e z. Formula: ηN,inst = max(NEd/Nb,y,Rd, NEd/Nb,z,Rd)

⚖ Pressoflessione - Sezione

Interazione N-M per sezioni classe 1-2 (formula quadratica) o 3-4 (formula lineare). Applica riduzione MN per N.

⚖ Instabilità Flessione + Compressione

Interazione globale con coefficienti ky, kz e μy, μz secondo Annex B EC3.

⚖ Svergolamento / Flesso-Torsionale

Momento critico elastico Mcr con formula completa di rigidezza torsionale e ingobbamento. Coefficiente χLT.

Esempio di Coefficiente di Sfruttamento

VerificaFormulaDescrizioneEsito
Forza normale - sezioneη = NEd / Npl,RdSemplice compressione/trazioneOK se η ≤ 1.0
Forza normale - instabilitàη = max(NEd/Nb,y,Rd, NEd/Nb,z,Rd)Instabilità eulerianaOK se η ≤ 1.0

3. Classificazione della Sezione

La classificazione della sezione è il primo passo del calcolo e determina il metodo di verifica applicabile. Le sezioni sono classificate in 4 classi in base alla snellezza degli elementi compressi.

Parametro di snellezza
ε = √(235 / fy) = √(235 / fy)
Classificazione delle Sezioni - Limiti EC3-1-1

Formula di Classificazione per Sezioni I

Ala compressa (c = (b - tw)/2)
λala = c / tf
Anima (c = h - 2·tf)
λanima = c / tw

Limiti di Classe per Profili I (ε = √(235/fy))

ElementoClasse 1Classe 2Classe 3Classe 4
Ala compressaλ ≤ 9ελ ≤ 10ελ ≤ 14ελ > 14ε
Anima compressioneλ ≤ 33ελ ≤ 38ελ ≤ 42ελ > 42ε
Anima flessioneλ ≤ 72ελ ≤ 83ελ ≤ 124ελ > 124ε
⚠ Nota: La classe della sezione è determinata dal valore massimo tra ala e anima, e può differire per compressione pura, flessione pura e pressoflessione.

Sezione Efficace per Classe 4

Snellezza della piastra (EC3-1-5)
λp = (c/t) / (28.4 · ε · √k)
Fattore di riduzione
ρ = 1 se λp ≤ 0.673, altrimenti ρ = (λp - 0.165) / λp²

5. Metodologia di Calcolo e Normative

5.1 Resistenze della Sezione

Resistenza plastica a sforzo assiale (EC3-1-1 6.2.3)
Npl,Rd = (Aeff · fy) / γM0
Momento resistente (EC3-1-1 6.2.5)
Mc,y,Rd = (Wres,x · fy) / γM0

5.2 Instabilità Flessionale

Carico critico euleriano
Ncr = π² · E · I / l0²
Snellezza adimensionale
λ¯ = √(Npl,Rd / Ncr)
Coefficiente Φ (EC3-1-1 6.3.1)
φ = 0.5 · [1 + α(λ¯ - 0.2) + λ¯²]
Coefficiente di riduzione
χ = 1 / [φ + √(φ² - λ¯²)]
Resistenza a compressione con instabilità
Nb,Rd = χ · (Aeff · fy) / γM1

5.3 Curve di Instabilità

Curvaα (imperfezione)Sezioni laminateSezioni saldate
a00.13Acciai ad alta resistenza-
a0.21IPE, HEA (h/b < 1.2)Box saldati (t ≤ 40mm)
b0.34IPE, HEA (h/b ≥ 1.2)Box saldati (t ≤ 40mm)
c0.49HEB, HEM, CanaliBox saldati (t > 40mm)
d0.76-Sezioni speciali

5.4 Svergolamento (Lateral Torsional Buckling)

Momento critico elastico (EC3-1-1 6.3.2)
Mcr = C1 · (π²E·Iz)/(k·L)² · √[(G·It + π²E·Iw/(kw·L)²)/(E·Iz)]
Resistenza a svergolamento
Mb,Rd = χLT · Mref dove Mref = Mcy,Rd · γM0 / γM1
ⓘ Nota Tecnica: Il coefficiente C1 tiene conto della forma del diagramma del momento flettente. Valori tipici: 1.00 (uniforme), 1.13 (triangolare), 0.90 (doppia curvatura).

8. Limitazioni e Note Tecniche

⚠ Vincoli

L'applicazione non può dedurre automaticamente i vincoli reali dalla sola sezione trasversale. L'utente deve impostare esplicitamente i preset k, kw e C1 in base alle condizioni di vincolo reali della struttura.

⚠ Sezioni cave

Per le sezioni CHS, RHS, SHS e alcune famiglie speciali, il modello di svergolamento è meno significativo. La verifica principale rimane forza normale, pressoflessione e instabilità flessionale.

ⓘ Taglio

La verifica a taglio non è attualmente implementata nella verifica diretta. Per la combinazione flessione-taglio si applica la riduzione del momento resistente secondo EC3-1-1 6.2.8.

ⓘ Warping

La costante di ingobbamento Iw non è sempre disponibile nel profilario. In assenza, si assume un valore minimo per garantire la convergenza del calcolo.

Bug Noti

❌ Limitazione documentata: Il coefficiente C1 per il momento critico è implementato solo con preset semplificati. Per distribuzioni di momento complesse (non lineari) si consiglia l'uso del modulo LTBeam-style che calcola C1 automaticamente dalla discretizzazione.

Raccomandazioni d'Uso

  • Verificare sempre i vincoli: I coefficienti k e kw devono essere impostati correttamente in base alle condizioni di vincolo reali.
  • Controllare la classe: Per sezioni dubbie, verificare i limiti nella tabella di classificazione.
  • Usare LTBeam-style: Per travi con carichi distribuiti o ritegni intermedi, preferire il modulo avanzato.
  • Esportare e archiviare: Salvare sempre l'input JSON e la sintesi CSV per la documentazione del progetto.
